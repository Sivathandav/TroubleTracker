# Hero Image Instructions

## Required Image
Please export the hero image from Figma and save it as:
`client/src/assets/images/hero-image.jpg`

## Image Specifications
- The image should show a person working on a laptop with analytics/charts
- Dimensions: Should be high resolution (at least 1920x1080)
- Format: JPG or PNG
- The image is currently using a placeholder URL

## How to Export from Figma
1. Open the Figma design
2. Select the hero image on the login page
3. Click "Export" in the right panel
4. Choose JPG or PNG format
5. Export at 2x or 3x resolution for best quality
6. Save as `hero-image.jpg` in this folder

## Current Placeholder
The CSS is currently pointing to: `../../assets/images/hero-image.jpg`

Once you add the actual image, the app will automatically use it.
